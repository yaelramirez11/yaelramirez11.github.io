Mi trabajo

Descripción general del proyecto: Alrededor de los EE.UU.

Aplicación web desarrollada con React, centrada en la construcción de una interfaz moderna y responsiva. El proyecto prioriza la organización del código, la reutilización de componentes y una navegación clara entre vistas. Se utilizan APIs para la gestión de datos, JavaScript como base del desarrollo y principios de Programación Orientada a Objetos (POO), aplicando buenas prácticas de desarrollo frontend para lograr una experiencia de usuario intuitiva, con especial atención al diseño, la usabilidad y el mantenimiento del proyecto.

Funcionalidades

  • Uso de Git y GitHub para el control de versiones y el trabajo organizado del código.
  • Desarrollo de la interfaz con HTML y CSS modernos, enfocados en un diseño estético, responsivo y consistente.
  • Implementación de Programación Orientada a Objetos (POO) en JavaScript, aplicando principios de organización y reutilización.
  • Uso de clases en JavaScript para estructurar la lógica del frontend.
  • Integración de la API de Tripleten para el almacenamiento y gestión de datos.

Logros

  • Implementación de formularios interactivos para la edición del perfil de usuario y la creación de nuevas tarjetas
  • Desarrollo de un sistema de autenticación de usuarios que permite modificar la foto de perfil
  • Control de permisos para la eliminación de tarjetas personales, garantizando que solo el usuario propietario pueda eliminarlas
  • Gestión del estado de “me gusta”, permitiendo a cada usuario guardar sus interacciones de forma individual
